# Asset attribution

## Pokémon type symbols (`types/`)

The type symbols are a **fan-made** ("Pokémon TCG — all type symbols") set
shared on Reddit:

- Source: <https://www.reddit.com/r/pokemon/comments/1p8mv1f/pokemon_tcg_trading_card_game_all_type_symbols/>
- Creator: u/RestorerForer (<https://www.reddit.com/user/RestorerForer/>)
- Retrieved: 2026-06-28

The single source image (`pokemon-tcg-trading-card-game-all-type-symbols-fanmade-to-v0-rekm4sfp814g1.webp`)
is a 5×5 grid. Each symbol was cropped into its own square logo (transparent
corners, tangent to the circle) and sorted into:

- `types/canonical/` — the 11 official Pokémon **TCG energy types**:
  Grass, Fire, Water, Lightning, Psychic, Fighting, Darkness, Metal, Dragon,
  Fairy, Colorless.
- `types/fanmade/` — the remaining non-TCG symbols.
